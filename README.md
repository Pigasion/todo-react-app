# Todo List (React)

Обновлённое веб-приложение, перестроенное на React. Поддерживает добавление, удаление и отметку выполнения задач, а также скролл к 1-ой невыполненной задачи.

В проекте используются функциональные компоненты и хуки. Состояние приложения синхронизируется с localStorage для сохранения списка задач между сессиями.

**Технологии**: React, CSS, JavaScript (ES6+).
